## 七、下午关键技术问题与解决

| 问题 | 根因 | 解决 |
|------|------|------|
| Spark 作业不显示在 UI | `local[2]` 模式不经过 Master | 改 `spark://master0:7077` + `--conf spark.cores.max=1` |
| 大屏练度不滚动 | foreachPartition 批量 LPUSH 后一次性 LTRIM | 逐条 LPUSH+LTRIM（匹配 Python 行为） |
| Satisfaction 不更新 | 旧 consumer group 从 earliest 回放 34,848 条 | `spark-satisfaction-v2` + `kafka-consumer-groups --reset-offsets` |
| `NotSerializableException` | Model 类未实现 Serializable | CharacterBuildRecord/WeaponChoice/ArtifactSetChoice/SatisfactionRecord |
| `NoClassDefFoundError: commons-pool2` | Jedis 连接池依赖缺失 | 下载 commons-pool2-2.4.2.jar |
| Satisfaction 抢 2 Core，Build 无资源 | Spark 默认分配所有可用 Core | `--conf spark.cores.max=1` |
| 抽卡管道循环卡死 | `--loop` pipe 链下游退出后管道断裂 | shell `while true; do ...; sleep 300; done` |
| CPU 显示 loadavg（如 1.0）无法理解 | `/proc/loadavg` 是负载非百分比 | `load ÷ nproc × 100` 转真实使用率 |
| Redis 内存永远 0% | 677KB ÷ 256MB = 0.26% | 显示绝对内存（MB）+ key 数 |
| 监控"消费端 0 进程" | Python 消费端已停用 | `streaming_count`（Spark 作业数） |
| 头像跨机器访问消失 | CDN 防盗链（localhost 正常，IP 访问被拦） | 待解决：后端代理或换图源 |
